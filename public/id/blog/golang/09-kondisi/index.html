Kondisi

Kondisi pada pemrograman di gunakan untuk mengatur flow dari sebuah program, sama saja dengan pengambilan keputusan pada dunia nyata, misalnya saat kita puasa kita di perbolehkan makan setelah jam tertentu, nah jam tertentu inilah sebuah kondisi yang menentukan kita boleh makan atau tidak. Sama juga dengan pengambilan keputusan pada sebuah program, sebuah kode dieksekusi ketika kondisi yang diberikan terpenuhi. Ini biasa juga di sebut sebagai Control flow statements.

Go mempunyai 2 macam keyword untuk memilih kondisi, yakni:

  • if, else statement
  • switch case

if, else if, else statement

Berikut contoh penggunaan if, else statement pada golang

package main
import "fmt"

func main() {
	time := 16
	bukaPuasa := 18

	if time > bukaPuasa {
		fmt.Println("saatnya makannn...")
	} else {
		fmt.Println("masih belum boleh makan")
	}
}

Kalau kita eksekusi kode diatas, hasilnya adalah:

Yappp, karna kondisi time > bukaPuasa tidak terpenuhi, maka code yang di dalam else lah yang akan tereksekusi.

Coba sekarang kita ganti value time nya menjadi 20, agar kondisi time > bukaPuasa terpenuhi.

package main
import "fmt"

func main() {
	time := 20
	bukaPuasa := 18

	if time > bukaPuasa {
		fmt.Println("saatnya makannn...")
	} else {
		fmt.Println("masih belum boleh makan")
	}
}

hasilnya adalah:

Jika kita ingin menggunakan if,else untuk lebih dari 1 kondisi, kita bisa memakai keyword else if

Berikut contoh kodenya:

package main
import "fmt"

func main() {
	lampuLaluLintas := "HIJAU"

	if lampuLaluLintas == "MERAH" {
		fmt.Println("Berhenti")
	} else if lampuLaluLintas == "KUNING" {
		fmt.Println("Hati-hati")
	} else if lampuLaluLintas == "HIJAU" {
		fmt.Println("Jalan")
	} else {
		fmt.Println("Lampu tidak benar / mati")
	}
}

if, else if, else statement dengan inisialisasi

Saat menggunakan if statement pada golang kita juga di perbolehkan untuk melakukan inisialisasi value, begini contohnya:

package main
import "fmt"

func main() {
	if lampu := "Kuning"; lampu == "Kuning" {
		fmt.Println("Hati-hati")
	}
}

Hasil eksekusi kode diatas:

Perlu di ingat, bahwa variable lampu hanya bisa di panggil di dalam block if tersebut, di luar block variable lampu tidak dapat di kenali.

if, else if, else statement bercabang / nested

Dalam perkondisian, ada istilah nested if, nested if ini artinya ada if didalam if, berikut contoh kodenya:

package main
import "fmt"

func main() {

	nilai := 57

	if nilai > 80 {
		fmt.Println("A")
	} else if nilai > 60 {
		fmt.Println("B")
	} else {
		if nilai < 30 {
			fmt.Println("D")
		} else if nilai < 60 {
			fmt.Println("C")
		}
	}
}

Hasil eksekusi kode diatas:

Nested if kurang di anjurkan karna kodenya cenderung komplex.

if,else juga kurang di rekomendasikan, jika kondisinya terlalu banyak. Jika kondisinya banyak, lebih di anjurkan memakai switch-case yang akan kita bahas üòÉ.

switch-case pada golang

switch-case bisa kita pakai sebagai alternative if-else, jika kondisi alur program terlalu banyak. Berikut contoh kode penggunaan switch-case:

package main
import (
	"fmt"
)

func main() {
	lampuLaluLintas := "KUNING"

	switch lampuLaluLintas {
	case "MERAH":
		fmt.Println("Stop!!")
	case "KUNING":
		fmt.Println("Hati-hati!")
	case "HIJAU":
		fmt.Println("Jalann!")
	default:
		fmt.Println("Lampu tidak valid")
	}
}

Hasil eksekusi kode diatas:

switch-case dinilai lebih elegan, dan lebih mudah terbaca dibanding if-else jika kondisi yang akan di seleksinya banyak.

keyword switch digunakan untuk memberitahu program, mana variable yang akan kita seleksi

keyword case digunakan untuk memberitahu program kondisi yang akan di cek

keyword default digunakan untuk nilai default, jika tidak ada kondisi yang terpenuhi, sama saja dengan keyword else
